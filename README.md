# 🏦 Smart ATM Simulation System (Python)

## Overview

A **Smart ATM Simulation System** built using Python that mimics real-world ATM operations.
This project focuses on delivering a clean user experience with logical flow and additional creative features.
---

## Key Features

*  Secure PIN Authentication
*  Balance Checking
*  Deposit Functionality
*  Withdrawal System with Validation
*  Transaction History Tracking
*  Recursive Transaction Counter
*  Receipt Pattern Generator
*  Reward Points System

---

##  Technical Highlights

* Uses **functions and modular programming**
* Implements **recursion** for transaction counting
* Demonstrates **loop-based menu systems**
* Clean and structured **user interaction design**

---

## Tech Stack

*  Python 3
* Standard Libraries (`time`, `random`)

---

##  How It Works

1. User enters name and ATM PIN
2. Upon successful authentication, menu is displayed
3. User can perform:

   * Check balance
   * Deposit money
   * Withdraw money
   * View transaction history
4. System tracks all transactions dynamically
5. Displays receipt patterns and reward points after transactions
